A **web-based payroll management system** to manage employees, attendance, allowances, deductions, and payroll efficiently. Built with **PostgreSQL** as the database, **Supabase** for backend API calls, and a modern responsive **HTML/CSS/Tailwind** frontend.


## Features

### Employee Management
- Add, edit, and view employee details
- Assign jobs, departments, and roles

### Attendance Management
- Record employee attendance by month
- Track Present, Absent, and Leave counts
- Filter attendance by employee and month

### Allowance & Deduction Management
- Maintain master lists of allowances and deductions
- Assign allowances and deductions to employees dynamically
- View and manage employee allowances and deductions

### Payroll Management
- Auto-calculate payroll based on attendance, allowances, deductions, and basic salary
- View monthly payroll for all employees
- Generates net salary and tracks payroll records

### Authentication & Dashboard
- HR login verification
- Responsive dashboard with quick navigation
- Logout functionality

### UI/UX
- Modern interface using **Tailwind CSS**
- Fully responsive for desktop and mobile
- Background images, overlays, and smooth hover effects

---

## Technology Stack
- **Frontend:** HTML, Tailwind CSS, JavaScript
- **Backend:** PostgreSQL, Supabase RPC functions
- **Deployment:** Static frontend with Supabase backend
